User Guide
==========

Last updated: 23 February 2021, for *SBEMimage* 2020.12

This is the user guide for the acquisition software *SBEMimage*.

For support questions and general discussion, please use the `Image.sc forum <https://forum.image.sc/>`_, with the tag ‘`sbemimage <https://forum.image.sc/tag/sbemimage>`_’. For bug reports use `GitHub Issues <https://github.com/SBEMimage/SBEMimage/issues>`_.

The source code of *SBEMimage* is hosted on `github.com/SBEMimage <https://github.com/SBEMimage/SBEMimage>`_. The source files for this user guide are located in the folder `/docs <https://github.com/SBEMimage/SBEMimage/tree/master/docs>`_.
